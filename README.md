# Urban Heat Technical Manual Repository

This repository contains the full technical documentation for replicating the Urban Heat Risk and 3-30-300 Framework Analysis conducted for Nairobi as part of the Carnegie Mellon University capstone project in partnership with the World Bank Climate Finance and Economics Unit.

The goal of this repository is to provide a transparent and fully reproducible workflow that cities, GIS analysts, and development practitioners can use to apply the same methodology in any urban area.
